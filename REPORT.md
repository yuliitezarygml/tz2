# Звіт з розробки мобільного застосунку Android з таймером зворотного відліку

## Опис завдання
Створити мобільний застосунок під Android, який реалізує таймер зворотного відліку від 10 секунд після натиснення кнопки "Start". Інтерфейс користувача має бути створений у XML-шаблоні, а логіка — у Java-класі CountdownActivity.

## Хід роботи

### Крок 1: Підготовка проекту
- Використано існуючий проект Android у папці `c:\Users\admin\AndroidStudioProjects\MyApplication2`.
- Проект ініціалізований з Gradle, містить стандартну структуру для Android додатків.

### Крок 2: Створення інтерфейсу користувача
- Створено файл `app/src/main/res/layout/activity_countdown.xml` з XML-шаблоном:
  - Використано `RelativeLayout` як кореневий контейнер.
  - Додано `TextView` з ідентифікатором `time_display_box` для відображення часу (початковий текст "00:10").
  - Додано `Button` з ідентифікатором `startbutton` та текстом "Start".
- Оновлено файл `app/src/main/res/values/strings.xml`, додавши рядки:
  - `start`: "Start"
  - `_00_10`: "00:10"

### Крок 3: Написання логіки додатку
- Створено клас `CountdownActivity.java` у пакеті `com.example.myapplication`:
  - Розширює `Activity`.
  - Використовує `CountDownTimer` для відліку від 10 секунд.
  - У методі `onCreate` підключає XML-шаблон, знаходить елементи інтерфейсу та встановлює обробник натиснення кнопки.
  - Метод `showTimer` створює та запускає таймер, оновлюючи текст кожну секунду ("counting down: X") та виводячи "PRYVIT!" після завершення.
- Оновлено `AndroidManifest.xml`, змінивши основну активність на `CountdownActivity`.

### Крок 4: Збірка та тестування
- Виконано збірку проекту за допомогою команди `gradlew build -x test` у терміналі.
- Збірка пройшла успішно без помилок.
- Проект готовий до запуску на емуляторі або пристрої.

### Крок 5: Запуск додатку
- Для запуску відкрийте проект у Android Studio.
- Запустіть емулятор Android (наприклад, Pixel 4 API 30).
- Натисніть "Run" або Shift+F10 для встановлення та запуску додатку.
- Після запуску:
  - Екран показує "00:10" та кнопку "Start".
  - Натиснення кнопки запускає відлік: текст змінюється на "counting down: 10", потім зменшується до "counting down: 1".
  - Після завершення текст змінюється на "PRYVIT!".

### Крок 6: Створення документації
